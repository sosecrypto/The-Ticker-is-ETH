⚙️ 프로토콜 업데이트

- EIP-8037 관련 처리량과 상태 성장 사이의 균형을 맞추기 위한 연구가 진행됨. 비대칭 집계 함수를 도입하여 가스 가격 책정의 효율성을 높이고 네트워크 부하를 조절하려는 시도임.

📋 EIP 동향

- ERC-8172 제안됨. 검증된 에이전트가 메타데이터를 즉시 변경하는 것을 막기 위해 쿨다운 기간을 도입함. 이는 사용자에게 변경 사항을 확인하고 대응할 시간을 제공함.

- VOSA라는 새로운 프라이버시 패턴이 제안됨. 머클 트리 없이 일회용 주소를 사용하는 UTXO 방식을 EVM에 적용하여 단순하면서도 규정 준수가 용이한 익명 거래 구현을 목표로 함.

🔬 연구 하이라이트

- 계정 기반 블록체인을 위한 익명 기밀 결제 프로토콜인 Anonymous PGC 연구가 발표됨. Twisted ElGamal 기술을 고도화하여 기존 솔루션의 한계를 극복하고 결제 보안성을 강화함.

- 실행 API v1.0.0 버전 관리 초안이 검토됨. 칸쿤 업그레이드까지의 71개 메서드를 포함하며 에러 코드 표준화와 테스트 커버리지 확대를 통해 RPC 서비스의 안정성을 도모함.

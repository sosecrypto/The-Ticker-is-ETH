오늘의 이더리움 소식(2026-01-30)

💰 ETH: $2,721 | 📊 ETH.D: 11.4% | 🔒 Staking: 30.5%

⚙️ 프로토콜 업데이트

- EIP-8142 Block-in-Blobs 제안됨. 실행 데이터를 블롭(대용량 데이터 저장소)에 포함하여 네트워크 효율성 개선 시도함.

- EIP-7807 SSZ 실행 블록 제안됨. 데이터 구조를 기존 RLP에서 SSZ(효율적 직렬화 방식)로 전환하여 검증 속도 향상 목표함.

- Glamsterdam 업그레이드 관련 가스비 재조정 논의 진행중. EIP-7904 및 EIP-8037 제안의 현재 상태 점검함.

- 이더리움 재단은 확장성과 성능을 보장하는 공격적인 로드맵 이행을 위해 완만한 긴축 재정 기간에 진입함.

📋 EIP 동향

- EIP-8141 프레임 트랜잭션 제안됨. 트랜잭션 유효성과 가스 결제를 추상적으로 정의할 수 있는 새로운 트랜잭션 유형임.

- ERC-8004가 실제 서비스에 통합됨. 이더리움을 AI 에이전트 간의 데이터 및 리스크 관리 조정 레이어로 활용하기 위한 표준임.

- ERC-7730 구조화된 데이터 클리어 사이닝 포맷 논의됨. 지갑 사용자가 서명 시 트랜잭션 내용을 더 명확히 확인 가능함.

- EIP-8037 상태 생성 가스 비용 인상 제안에 따른 실패 모드 및 가스 스케일링 영향에 대한 분석 연구가 진행됨.

- EIP-8141 프레임 트랜잭션의 상태 필드 번호 수정 사항이 깃허브 저장소에 반영됨.

🔬 연구 하이라이트

- 1년치 활성 상태만 유지하는 실험 진행됨. 상태 만료(State Expiry)가 실제 노드 운영에 미치는 구체적 데이터를 분석함.

- 애플리케이션 제어 실행(ACE) 연구됨. 앱이 트랜잭션 순서를 직접 제어하여 온체인 거래 효율을 높이는 메커니즘을 다룸.

- EIP-8037 도입 시 사용자 예산 배분 변화 분석됨. 가스 한도 증대 시 블록 공간 활용도가 저하될 수 있음을 경고함.

- PSE(개인정보보호 연구팀)에서 모바일 GPU를 활용한 프라이버시 보호 로드맵 공유함. 영지식 증명 가속화가 핵심임.

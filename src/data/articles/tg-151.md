[Arbitrum, BoLD 도입 직전]

아비트럼이 permissionless validator 시스템인 BoLD를 도입하는 거버넌스 프로포절을 올려, 현재 DAO의 투표를 받고 있습니다.

기존 아비트럼은 총 14명의 정해진 밸리데이터들만이 상태 루트를 제출하고 챌린지를 걸 수 있는 권한을 가지고 있었습니다. BoLD가 도입된다면, 누구나 아비트럼 노드를 돌려 체인을 검증하고, 챌린지를 걸 수 있게 됩니다. 물론 챌린지를 걸려면 최소 3,600 ETH가 있어야 하지만요.

BoLD가 도입되면, 아비트럼은 Stage 2에 조금 더 가까이 갈 수 있습니다. 남은 과제는 다음과 같습니다.

⁃ 챌린지 상황에서 Security Council의 개입이 ‘온체인에서 증명 가능할 때만’ 이뤄지도록 보장하는 것 (e.g. multi-proof 도입 등)

⁃ 사용자들의 exit window, 즉 컨트랙트 업그레이드 딜레이를 30일 이상으로 늘리는 것

만약 해당 작업이 빠르게 이루어질 수 있다면, 아비트럼은 메이저 롤업 중 최초로 Stage 2에 도달하여, 이더리움의 보안을 온전히 상속받게 될 수 있습니다.

이더리움 2026년 프로토콜 우선순위 업데이트

2025년 주요 성과: Pectra(5월)와 Fusaka(12월) 두 차례 메이저 업그레이드를 성공적으로 배포했습니다. 

EIP-7702(EOA의 스마트 컨트랙트 코드 실행), PeerDAS(블롭 데이터 샘플링으로 대역폭 절감 및 블롭 용량 8배 증가), 가스 리밋 30M→60M 인상, 히스토리 만료 등이 포함됩니다.

2026년 3개 트랙 재편:

1. Scale (Ansgar, Marius, Raúl): 기존 L1·블롭 스케일링을 통합. 가스 리밋 100M+ 목표, ePBS, zkEVM 어테스터 클라이언트, 상태 스케일링

2. Improve UX (Barnabé, Matt): 네이티브 계정 추상화(번들러/릴레이어 없는 스마트 컨트랙트 지갑)와 L2 상호운용성에 집중. 포스트-양자 대비도 연계

3. Harden the L1 (Fredrik, Pari, Thomas): 신규 트랙. 보안(포스트-양자, 트러스트리스 RPC), 검열 저항성(FOCIL), 네트워크 복원력 및 테스트 인프라

다음 업그레이드: Glamsterdam(2026년 상반기), Hegotá(2026년 하반기) 예정.

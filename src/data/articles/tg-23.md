이더리움 Pectra & Fusaka Upgrade 톺아보기

by 노민섭, 허윤정 of Decipher

이 아티클은 이더리움 네트워크의 장기 로드맵에서 중요한 단계인 Pectra와 Fusaka 업그레이드에 대해 다룹니다. 두 업그레이드는 UX와 확장성 개선을 목표로 하는 주요 EIP들을 포함하고 있으며, Pectra는 내년 상반기, Fusaka는 하반기에 예정되어 있습니다. 주요 제안 중 Pectra의 EIP-7702는 EOA 계정에 스마트 컨트랙트 기능을 부여해 다양한 작업을 가능케 하고, EIP-7623은 calldata 가스 소모량을 조정해 블록 사이즈 효율성을 높입니다. Fusaka의 EIP-7692는 EVM Object Format을 통해 스마트 컨트랙트 구조와 안전성을 개선하며, EIP-7594는 DAS 프로토콜을 통해 데이터 가용성을 확보하여 확장성을 강화합니다. 이더리움은 안전성을 유지하면서도 지속적인 개선을 추진하고 있으며, 이러한 단계적 업그레이드가 장기적으로 이더리움의 경쟁력을 높일 것으로 기대됩니다.

- Pectra Upgrade

  - EIP-7702: Set EOA account code

  - EIP-7623: Increase calldata cost

  - EIP-7251: Increase the MAX_EFFECTIVE_BALANCE

  - EIP-6110: Supply validator deposit on chain

- Fusaka Upgrade

  - EIP-7692: EVM Object Format Meta

  - EIP-7594: PeerDAS — Peer Data Availability Sampling

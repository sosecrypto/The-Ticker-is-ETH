🔹리스크 제로, 44초만에 이더리움 블록 ZK 블록 증명

대표적인 ZK 프로젝트 리스크 제로(RISC Zero)에서 R0VM 2.0 공개했습니다.

이와 함께 ZK 기술이 이론적 단계를 넘어 실사용 가능한 시대에 진입했다고 선언했습니다.

R0VM 2.0은 1.0과 비교해서 다음과 같은 성능 향상이 있었습니다.

1. 증명 속도 향상

-이더리움 블록 증명 시간: 35분 → 44초

-7월까지 실시간 증명(12초) 달성 목표

2. 증명 비용 절감

-베이스에서 $0.0107 → $0.0022 (5배 감소)

-어떤 롤업도 ZK 롤업으로 전환 가능

3. 메모리 확장

-한 번에 증명 가능한 메모리 크기: 192mb → 3GB

-단일 증명으로 수천 개 블록 증명 가능

4. 프리컴파일 추가

-페어링 연산 중심 작업 최대 13배 속도 향상

-EVM 실행 및 이더리움 합의 증명용 주요 프리컴파일 모두 지원

5. 형식 검증 기반 보안 강화

-RISC-V 기능 123개 중 122개가 결정론적이라는 것을 수학적으로 증명

증명 속도는 이더리움 블록으로, 증명 비용은 베이스 블록으로 얘기하는 게 좀 쎄하고 실제로 ZK가 실사용할 수준인지 모르겠지만, 많은 성능 개선이 있었던 건 맞는 것 같습니다.

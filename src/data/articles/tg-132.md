ETH Rangers Program

Incentivizing Public Goods Security Work for the Ethereum Ecosystem

이더리움 재단(EF)에서 보안 관련 공공 기여 활동에 대한 그랜트 프로그램을 발표했습니다.

What type of work is considered Public Goods Security work?

다음과 같은 활동들을 통해 생태계 보안을 위해 기여하는 개인들을 지원할 예정입니다.

- 취약점 리서치

- 보안 관련 컨텐츠 제작

- 보안 사고 사후 분석 및 오딧

- 개발자 도구 및 프레임워크 개발

내년 1월부터 시작해서 6월에 리포트 제출하면서 활동을 마무리하고 개인별로 총 $25,000의 지원금을 지급합니다.

Who is running this program?

- EF : 이더리움 재단에서 프로그램을 주관하며 필요한 자금도 지원합니다

- Secureum : 스마트 컨트렉트 보안에 관한 부트캠프를 운영하고 보안 관련해서 교육 컨텐츠도 많이 발간하고 있습니다

- The Red Guild : 다양한 보안 연구와 함께 Damn Vulnerable Defi (링크)라는 스마트 컨트렉트 취약점을 직접 경험해 볼 수 있는 사이트를 운영하고 있습니다

- Security Alliance(SEAL) : 패러다임에서 리서치로 일하고 있으며 화이트해커로 활동하고 있는 samczsun에 2023년 8월에 시작한 팀으로 취약점이 발견될 경우, 혹은 긴급한 보안 사고가 발생했을 경우 도움을 청할 수 있는 텔레그램 핫라인을 운영하고 있습니다 (아마 블록체인 보안 관련해서는 가장 전문성 있는 팀이 아닐까 생각하고 있습니다)

여담으로 이더리움의 연대기를 기록한 책 The Cryptopians 에서 2016년의 The DAO 해킹 사태 때 보안 전문가들이 어떻게 대응했고 어떤 고초를 겪었는지 잘 나와있는데요, 그 대목을 흥미진진하게 읽었던 기억이 나네요

파캐스터 Frames v2 발표

Farcaster는 발견(discovery) 측면에서는 훌륭했지만 실제 사용성은 떨어졌습니다.

사용자가 피드에서 앱을 발견하면 링크 클릭 → 페이지 로딩 → 지갑 연결이라는 과정을 거쳐야 했습니다. 

데스크톱에서는 가능했지만 모바일에서는 고급 사용자조차도 거의 사용이 불가능했습니다.

Frame v1 한계점

1. 작은 크기와 제한된 상호작용으로 많은 앱 구현 불가

2. 이미지로 렌더링되어 생성 속도가 느림

3. 일시적이어서 사용자가 다시 찾기 어려움

이러한 문제를 해결하기 위해 대화형 애플리케이션, 온체인 거래, 사용자 알림을 지원하는 새로운 프레임 표준이 필요하게 되었습니다. 

새로운 Frames v2의 주요 개선 사항

1. 화면 및 상호작용

- 전체 화면 애플리케이션으로 확장

- 브라우저에서 렌더링 가능한 모든 인터페이스 구현 가능

- 더 이상 이미지 렌더링에 의존하지 않아 속도 개선

2. 사용자 경험

- 피드에서 컴팩트한 형태로 표시 (이미지 + 버튼)

- 앱 내 브라우저에서 바로 실행

- 재방문/재사용이 가능하도록 영구적 접근성 제공

3. 개발자 기능

- 사용자 컨텍스트 정보 접근 (Farcaster 계정, 지갑 정보 등)

- 앱과 지갑 간 직접적인 상호작용 가능

사용자에게 알림 전송 기능

- SDK를 통한 다양한 액션 구현 가능

4. 통합성

- 온체인 트랜잭션 직접 처리

- 지갑 연결 프로세스 간소화

- 모바일과 데스크톱 모두에서 일관된 사용자 경험 제공

이를 통해 새로운 형태의 소셜 애플리케이션 개발이 가능해지고, 사용자들은 더 나은 경험들을 하게 될 것.

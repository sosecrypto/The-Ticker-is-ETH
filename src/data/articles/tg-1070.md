🔴 옵티미즘의 포스트-퀀텀 로드맵

옵티미즘이 양자컴퓨터 시대를 대비한 10년짜리 로드맵을 발표했습니다.

1. ECDSA EOA 10년 내 폐기

- 2036년 1월까지 ECDSA 기반 EOA 트랜잭션 지원 중단

- 모든 EOA는 포스트-퀀텀 스마트 컨트랙트 계정으로 키 관리 위임 필요

2. 유저 지갑 마이그레이션

- EIP-7702를 활용해 EOA가 스마트 계정에 서명 권한 위임

- 주소나 잔액 이동 없이 키 관리 방식만 업그레이드 가능

- 구체적인 PQ 서명 스킴은 아직 미정 (NIST 표준 격자 기반 서명 등 검토 중)

3. 인프라 업그레이드

- L2 시퀀서와 배치 서브미터도 PQ 서명으로 전환

- 슈퍼체인 전체에 일관된 업그레이드 롤아웃

4. 이더리움 레이어

- 현재 이더리움은 BLS 서명(밸리데이터)과 KZG 커밋먼트(DA/블롭)에 의존

- 장기적으로 이더리움도 PQ 알고리즘으로 전환 필요하다는 입장

- EF와 이미 협의 중

유저들이 당장 뭔가 할 필요는 없고, 10년에 걸쳐 점진적으로 마이그레이션 가이드와 툴링이 제공될 예정이라고 하네요

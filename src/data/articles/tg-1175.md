이더리움의 장기 로드맵 (Strawmap)

오늘 이더리움의 장기 로드맵 Strawmap이 Justin Drake의 트위터를 통해 발표되었습니다.

간략하게 요약하자면 다음과 같습니다.

1. Fast L1 : 트랜잭션이 블록에 포함되는것과 체인의 Finality를 모두 초 단위로 단축

2. Gigagas L1 : zkEVM을 활용해 이더리움 L1의 처리량을 1 gigagas/sec까지 향상 ( 10K TPS 수준)

3. Teragas L2 : L2를 위한 DA처리량을 1GB/sec까지 향상 (~10M TPS)

4. Post-quantum L1 : 해시함수 기반 암호 알고리즘을 통해 양자컴퓨팅 이후에도 수세기동안 영구적으로 지속 가능한 보안 달성

5. Private L1 : Shielded transfer(프라이빗 전송)을 L1 프로토콜 수준에서 구현

이는 6개월 간격의 총 7개의 포크를 통해 순차적으로 구현될 예정입니다. 즉, 다음번 Glamsterdam 포크부터 3.5년 뒤인 2029년까지 완성을 목표로 합니다.

이 로드맵이 완성된다면, 2029년 이더리움은 다음과 같은 모습의 체인이 됩니다.

- L1의 TPS 10K수준

- L2의 처리량 10M TPS

- 블록타임 1초까지 단축

- 초 단위의 최종성

- 양자 저항

- L1에서의 프라이빗 트랜잭션이 가능

기존 로드맵(Merge/Surge/Scourge/Verge/Purge/Splurge)이 "무엇"을 다루었다면, strawmap은 "어떤 순서로, 어떤 포크에"를 구체적으로 다룬다고 할수 있습니다.

그리고 당연히 이를 달성하기 위해 탈중앙화의 희생은 없을 것입니다.

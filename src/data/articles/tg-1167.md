⚙️ 프로토콜 업데이트

- 이더리움 재단이 보유 자산 중 7만 ETH 스테이킹 시작함. 발생 수익은 재단 운영 예산으로 환원되어 생태계 장기 지원에 활용될 예정임

- FOCIL(포크 선택 기반 포함 리스트) 사양 업데이트 및 개발 현황 논의됨. 검열 저항성 강화를 위한 핵심 기술임

- EIP-7928 블록 수준 액세스 리스트 최적화 진행중. 블록 처리 효율을 높이기 위한 항목 제한 및 사양 명확화 작업 포함됨

- 멤풀 암호화(EIP-8105) 논의 시작됨. 트랜잭션 기밀성 유지를 위한 비콘 체인 RPC API 및 키 공개 메커니즘 검토중

📋 EIP 동향

- EIP-8105(멤풀 암호화) 제안 상태 검토됨. 트랜잭션이 블록에 포함되기 전까지 내용을 숨겨 앞지르기 공격을 방지하는 목적임

- EIP-7928 업데이트 진행중. 상태 전이 함수와 액세스 리스트 간의 관계를 명확히 하고 블록당 최대 항목 수 확인 로직 단순화함

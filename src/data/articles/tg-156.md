🔷 이더스캔 2024년 리캡 

이더스캔이 작업한 24가지 (전체 리스트는 펼쳐보세요!)

1. Explorer as a Service

> SVM Rollup을 포함한 8개 explorers 제작

> 2개의 L3 explorers, 3개의 L1 explorers

> Explorer 전체 리스트

2. Blockscan Multichain Explorer (Beta)

여러 개 체인에 있는 자산들을 보여주는 것 (출처)

3. 이더스캔 카드

신용카드 개념이 아니라 에어드랍 클레임이나 Revoke 등의 기능을 이더스캔에서 카드 형태로 보여주는 서비스 (출처)

4. Blobs .oO

블롭 데이터를 보여주는 기능 (출처)

5. Etherscan API V2: Multichain (Beta)

50개 이상의 EVM 체인의 쿼리 엔드포인트를 단일 API키로 관리할 수 있는 서비스 (출처)

6. Account Abstraction Transactions (Beta)

계정추상화 관련 데이터를 볼 수 있는 서비스 (출처)

7. Input Data Decoder (출처)

8. Diamond Proxy Support (Beta) (출처)

9. 실험적 기능: Transaction Action (출처)

10. Stylus(아비트럼의 스타일러스) Contract Verification (출처)

11. Domain Name Support (출처)

12. Pending MEV-Protected Transactions (출처)

13. Extended NatSpec 지원 (출처)

14. Gas Tracker 소수점 지원 (출처)

15. Code Reader 엡데이트 (출처)

16. Token & Method 강조 기능 (UX 업데이트) (출처)

17. Converter Tools (출처)

18. Private Name Tag Groups (출처)

19. More Custom Errors (정확한 오류 표시로 개발자 디버깅이 쉬워짐) (출처)

20. 검색 결과 페이지 (UX 업데이트) (출처)

21. 토큰 이체에 대한 추가적인 안전장치 (출처)

23. ERC-20, ERC-721, ERC-1155 토큰 표준 체커 (출처)
